
### Initial directory structure

```

LibreChat-Azure-Deployment/
├── 1-setup-librechat.ps1          # Initial setup and configuration
├── 2-deploy-infrastructure.ps1     # Create Azure resources
├── 3-build-and-deploy.ps1          # Build images and deploy apps
├── 4-verify-deployment.ps1         # Verification and testing
├── config/
│   ├── .env                        # Environment variables
│   ├── librechat.yaml             # LibreChat configuration
│   └── Dockerfile                 # Container image definition
└── README.md                      # This guide

```
</br>
</br>


### Run individual scripts sequentially from #1 to #4

```
# Step 1: Setup LibreChat
.\1-setup-librechat.ps1 -ApimSubscriptionKey "your-apim-key" -ApimEndpoint "https://your-apim-endpoint"

# Step 2: Deploy Infrastructure
.\2-deploy-infrastructure.ps1 -ResourceGroup "librechat-rg" -Location "eastus"

# Step 3: Build and Deploy Apps
.\3-build-and-deploy.ps1 -ApimSubscriptionKey "your-apim-key" -ApimEndpoint "https://your-apim-endpoint"

# Step 4: Verify Deployment
.\4-verify-deployment.ps1

```

</br>

### OR

</br>

### Run and deploy everything at once

```
# Deploy everything at once
.\deploy-all.ps1 -ApimSubscriptionKey "your-apim-key" -ApimEndpoint "https://your-apim-endpoint"
```
